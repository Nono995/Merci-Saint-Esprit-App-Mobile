# Configuration API Bible

## API Abibliadigital - Bible Complète Gratuite

✅ **Aucune configuration nécessaire!**

L'app utilise maintenant l'API Abibliadigital qui est:
- ✅ **100% gratuite**
- ✅ **Pas de clé API requise**
- ✅ **Bible complète** (66 livres, tous les chapitres)
- ✅ **Louis Segond en français**
- ✅ **Pas de limite de requêtes**

## Comment ça marche

1. L'app charge automatiquement les chapitres depuis l'API
2. Si l'API échoue, fallback sur données locales
3. Aucune configuration nécessaire

## API Utilisée

- **Service**: Abibliadigital (https://www.abibliadigital.com.br)
- **Bible**: Louis Segond (LSG)
- **Endpoint**: `https://www.abibliadigital.com.br/api/verses/lsg/{book}/{chapter}`
- **Gratuit**: Oui (illimité)

## Fonctionnalités

✅ Bible complète (66 livres)
✅ Tous les chapitres disponibles
✅ Recherche de versets
✅ Fallback local si pas de connexion
✅ Partage WhatsApp
✅ Marquage de passages

## Livres disponibles

**Ancien Testament**: Genèse, Exode, Lévitique, Nombres, Deutéronome, Josué, Juges, Ruth, 1-2 Samuel, 1-2 Rois, Psaumes, Proverbes, Ecclésiaste, Cantique, Ésaïe, Jérémie...

**Nouveau Testament**: Matthieu, Marc, Luc, Jean, Actes, Romains, 1-2 Corinthiens, Galates, Éphésiens, Philippiens, Colossiens, 1-2 Thessaloniciens, 1-2 Timothée, Tite, Hébreux, Jacques, 1-2 Pierre, 1 Jean, Apocalypse...

## Utilisation

1. Lancez l'app: `npm start`
2. Allez dans Bible
3. Sélectionnez un livre
4. Sélectionnez un chapitre
5. Les versets se chargent automatiquement!
